# Agent instructions

This repository is a personal skill library. Each subdirectory under `skills/`
contains a `SKILL.md` describing a reusable behavior. Tools that follow the
`AGENTS.md` convention (Codex, cursor-agent, ...) should treat each `SKILL.md`
as available guidance and apply the relevant one when its `description` matches
the current task.

Skills available in this library are listed under `skills/`.
